# Portfólio Pessoal

Este projeto consiste no desenvolvimento de um portfólio online (currículo digital), criado como atividade acadêmica utilizando apenas tecnologias base da web.

---

## Tecnologias Utilizadas

* HTML5
* CSS3 (sem uso de frameworks)
* JavaScript (puro, sem frameworks)

---

## Funcionalidades

* Navegação entre páginas (menu fixo)
* Layout responsivo (adaptado para celular)
* Tema claro/escuro com persistência
* Formulário de contato com validação em JavaScript
* Interface moderna com efeitos visuais (hover e transições)

---

## Estrutura do Projeto

* `sobre.html` → Sobre mim
* `formacao.html` → Formação acadêmica
* `portfolio.html` → Projetos desenvolvidos
* `contato.html` → Formulário de contato
* `style.css` → Estilização do site
* `script.js` → Funcionalidades e interações
